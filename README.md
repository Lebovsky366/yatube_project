# Yatube
Социальная сеть для публикации личных дневников.

## Технологии
- Python 3.9
- Django 2.2.19

## Запуск проекта
- Создать виртуальное окружение
- Установить зависимости: `pip install -r requirements.txt`
- Перейти в папку `yatube/` и запустить сервер: `python manage.py runserver`
